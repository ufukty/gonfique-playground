Gonfique Playground

Enter the sample config file of yours and an optional Gonfique configuration to produce the mapping type. Later copy and paste the output into your project and call the ReadConfig function with the actula path during runtime provided via CLI arg.

This alpha version of Gonfique can make mistakes, check the results for correctness.

Playground loads with a Gonfique config for the sample config. Delete the provided Gonfique config following pasting your config sample in. The documentation is on the repository. It also houses the CLI version, which is better to integrate into your build pipeline.

Layout gives you one column per each 500 pixels of window width. So, it looks better on desktop.

Your inputs stays offline as Playground works inside the browser.
